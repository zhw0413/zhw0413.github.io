    反转链表

    算法 链表
    发布时间 : 2020-06-27 12:31

    内容：

    定义一个函数，输入一个链表的头节点，反转该链表并输出反转后链表的头节点。

    示例:

    输入: 1->2->3->4->5->NULL
    输出: 5->4->3->2->1->NULL

    思路：

    要想反转链表，可以新建一个新的链表codeList，将head中的数值到放入新链表codeList中，用于输出反转链表。为了保证head链表不被修改，还需要一个myhead = head，在myhead上进行操作，而不影响head的结构。

    对于链表为空或只有一个值时，可以直接输出链表。

    当链表不为空时，使用一个curNode链表节点进行过渡，将链表进行反转。将myhead的值赋给curNode。

    curNode.next = codeList.next ,codeList = curNode，这两步可以实现链表中值的反转，同时要记得将myhead的指针后移，指向下一个节点。

    当myhead.next=null时，要注意此时myhead中的最后一个节点还未传入codeList链表中，还需要借助一个链表进行如上的操作实现值的反转，方可结束操作。

    code:

    /**

 \* Definition for singly-linked list.
 \* public class ListNode {
 \*   int val;
 \*   ListNode next;
 \*   ListNode(int x) { val = x; }
 \* }
 */

class Solution {

  public ListNode reverseList(ListNode head) {
    ListNode codeList = new ListNode();
    ListNode myhead = head;

    if(head == null || head.next == null){
      return head;
    }
    codeList.val = myhead.val;
    myhead = myhead.next;
    while(myhead.next != null){
      ListNode curNode = new ListNode();
      curNode.val = myhead.val;
      curNode.next = codeList;
      codeList = curNode;
      myhead =myhead.next;
    }
    //myhead.next为空，但此时最后一个节点的值还未传入codeList中，需要借助curNode进行
    ListNode curNode = new ListNode();
    curNode.val = myhead.val;
    curNode.next = codeList;
    codeList = curNode;
    return codeList;

  }

}
